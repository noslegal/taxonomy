# iManage compatible CSV upload templates.

This repository contains iManage compatible CSV templates pre-loaded with the noslegal facets. iManage Insight+ customers can download these files, and use them to immediately populate taxonomy and lookup fields with the noslegal values.  

## Usage
Follow the guidance detailed in https://docs.imanage.com/cloud/Insightplus-cc-help/en/CSV_templates.html
